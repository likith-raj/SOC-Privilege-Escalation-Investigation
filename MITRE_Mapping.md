\# MITRE ATT\&CK Mapping



\## Attack Summary



A simulated Windows privilege escalation scenario was performed in a lab environment.



\---



\## MITRE ATT\&CK Techniques



| Activity | Technique |

|----------|-----------|

|Successful Login|Valid Accounts|

|New User Creation|Create Account|

|Added User to Administrators|Account Manipulation|



\---



\## Findings



The sequence of events indicates a potential privilege escalation scenario. In a production environment, this activity should be validated with the IT team to determine whether it was authorized or malicious.
