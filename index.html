SOC Case Study #01

Windows Privilege Escalation

Simulated privilege escalation attack investigation using Windows Event Logs and Splunk Enterprise.

Project Overview

This project demonstrates how a SOC analyst detects, investigates, and documents a privilege escalation attack using Windows Security logs collected in Splunk. The investigation covers log collection, event correlation, detection query development, and MITRE ATT&CK framework mapping.

Incident Summary

Key Findings

Investigation identified multiple failed authentication attempts followed by a successful login. Shortly afterwards, a new user account was created and added to the local administrators group.

Recommendations

Implement account lockout policies, enable logging for all administrative actions, and monitor for suspicious account creations in real-time.
